# Cheng-Hsuan Lee Portfolio Website

This is a GitHub Pages-ready static portfolio website for a robotics-focused Mechanical Engineering graduate application.

## Files

- `index.html` — homepage
- `quadruped.html` — main quadruped robot case study
- `frc.html` — FRC robotics experience
- `uav.html` — UAV propulsion and system estimation project
- `strain-analysis.html` — strain gauge and FEA project
- `reports.html` — technical reports library
- `resume.html` — resume snapshot page
- `assets/styles.css` — website styling
- `assets/script.js` — small helper script
- `images/` — place project images, GIFs, diagrams here
- `reports/` — place PDF reports and resume here

## How to publish on GitHub Pages

1. Create a GitHub repository, for example `portfolio`.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose the `main` branch and `/root` folder.
6. Save. Your site will publish at `https://YOUR_USERNAME.github.io/portfolio/`.
